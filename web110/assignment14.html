Final Project - How to Secure Your Home Network with pfSense and Pi-hole

My final project is titled "How to Secure Your Home Network with pfSense and Pi-hole". The objective of this project is to demonstrate how to effectively set up and manage a secure home network using pfSense as a firewall and Pi-hole as a network-wide ad and tracker blocker. This includes configuring pfSense for robust network security, setting up Pi-hole for DNS filtering, and integrating both for optimal performance. The project also focuses on documenting the setup process, testing network security, and creating a guide for replication. The scope includes researching best practices, installing and configuring software, testing the setup, and producing a video tutorial uploaded to YouTube. The end result will help others understand how to secure their home networks using pfSense and Pi-hole with clear steps.
